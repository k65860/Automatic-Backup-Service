# Automatic-Backup-Service
2021 멋쟁이사자처럼 at삼육대학교 아이디어톤

## 💻프로젝트 소개
- 웹에서 과제 작성 후 임시 저장 & 자동 재출이 가능한 서비스  
- 과제 자동 제출 On/Off 버튼 기능 구현  
- 사용자가 원하는 색상으로 테마 변경 가능  

## 개발 기간
21.06.28 ~ 21.07.10  

## 추진 배경
- e-class 에서 과제 제출을 잊은 경우  
- 제출 전날 한번 있는 알람에 불편함을 느낌
